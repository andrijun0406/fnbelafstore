
# F&B Elaf Store

A simple web application for managing Food & Beverage products and orders.

## Features
- Product listing by category
- Cart and checkout system
- Admin panel for managing products and orders

## Tech Stack
- PHP 7+
- MySQL
- Bootstrap (for UI)

   git clone https://github.com/andrijun0406/fnbelafstore.git
2. Import u952857351_fnb_store.sql into your MySQL database.
